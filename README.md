# Control-de-Flota-Semanal-Beta
Es una solución full-stack para propietarios y operadores de flotas de vehículos (transporte, renta, logística). Permite registrar bitácoras semanales, controlar costos de mantenimiento, gestionar siniestros y visualizar rentabilidad por unidad — todo sincronizado en la nube con soporte offline.
